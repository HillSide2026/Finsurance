# Marketing Agent Skills Matrix

This document defines the practical operating skills for each marketing agent.

Use it as the canonical skill reference for the system.

## 1. ICP Builder Agent

### Core Skills

- market segmentation for B2B SaaS
- jobs-to-be-done
- revenue-based prioritization
- founder and buyer psychology

### Advanced Skills

- trigger event mapping
- negative ICP design

### Core Skills

- intent inference
