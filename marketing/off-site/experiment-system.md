# Off-Site Experiment System

This document defines how off-site traffic sources should be tested.

## Objective

Identify which off-site source can produce qualified traffic into the on-site funnel with acceptable commercial signal quality.

## Operating Principle

No off-site source is strategic by default.

Each source starts as a hypothesis and must pass a test before it is scaled.

## Experiment Stages

### Stage 1

Micro-test.

Purpose:

- test message resonance
- test traffic quality
- test whether the channel can drive meaningful intent

Default limit:

- `14` days maximum
- organic effort allowed
- paid spend cap: `CAD 50`

### Stage 2

Paid pilot.

Purpose:

- validate repeatability
- measure early cost and signal quality

Default limit:

- `14` days maximum
- paid spend cap: `CAD 300`

### Stage 3

Expansion test.

Purpose:

- validate whether the channel is worth scaling

Default limit:

- requires Founder approval
- paid spend cap: `CAD 1000`

## Required Inputs For Every Test

Every channel test must define:

- channel
- hypothesis
- audience
- destination
- budget cap
- timebox
- success thresholds
- kill criteria

Use [channel-test-template.md](/Users/matthewlevine/Repos/Finsurance/marketing/off-site/channel-test-template.md).

## Default Success Threshold Categories

Each test should evaluate:

- qualified session volume
- primary CTA click-through
- pricing reach or service-route reach
- payment attempts
- qualified service leads

## Default Kill Conditions

Kill a channel test when one or more of the following is true:

- budget cap is reached with no meaningful commercial signal
- two consecutive tests fail traffic quality thresholds
- the channel produces traffic but almost no pricing reach or service lead intent
- the destination message does not match the channel audience

## Decision Outcomes

- `pass`
  - move to the next stage or scale carefully
- `iterate`
  - keep the channel hypothesis alive, but change message, audience, or destination
- `kill`
  - stop further spend and deprioritize the channel
