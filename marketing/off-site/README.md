# Off-Site

This folder is for marketing that happens off the product site.

## Purpose

This lane should operate as a channel experiment system.

Its job is to discover which off-site source can reliably produce qualified traffic into the on-site funnel.

## Current Hypothesis

- `linkedin/`

At the moment, LinkedIn is a working hypothesis, not a validated off-site source.
