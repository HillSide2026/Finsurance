# Marketing Contracts

This folder defines the schema-bound artifact contracts used by the marketing agent system.

## Purpose

These contracts exist so that:

- agents can plug into each other without ambiguity
- outputs are deterministic
- artifacts stay traceable
- Founder approval can happen against stable structures
